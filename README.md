# Currency Exchange

## Lýsing
Verkefni sem birtir gengi gjaldmiðla með því að nota ExchangeRate API. Síðan er byggð á Vite og notast við Tailwind CSS fyrir útlit.

## Tól notuð
- Vite
- Tailwind CSS
- ESLint
- Stylelint
- Jest

## Web API
- [ExchangeRate-API](https://www.exchangerate-api.com/)
